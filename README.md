# IELTS Academic 7+ Vocabulary Studio

A static single-page IELTS Academic study app. It runs from `index.html` with no server, database, or build step.

## Deploy

Upload the folder to any static host, or deploy it to Vercel as a static project.

## Local Preview

Open `index.html` directly in a browser, or run a simple static file server from this folder.

## Progress

Learner progress is stored locally in the browser under `ieltsAcademic7DeployProgressV1`.
